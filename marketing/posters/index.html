📸 어떻게 저장하나요?

  1. 포스터 카드의 "열기" 버튼 클릭 → 새 창에서 실제 크기로 열림
  2. Windows: Windows + Shift + S → 포스터 영역 드래그 → 클립보드 저장
  3. Mac: Command + Shift + 4 → 포스터 드래그 → 데스크탑 저장
  4. 고해상도: F12 → 반응형 모드 → 1080px × 1080px → 우클릭 "이미지 저장"
  5. 저장한 이미지 → 인스타그램 업로드
