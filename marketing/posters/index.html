📸 어떻게 저장하나요?

  1. 포스터 카드의 "열기" 버튼 클릭 → 새 창에서 실제 크기로 열림
  2. 화면에 포스터만 보이도록 F11 (전체화면) 눌러도 좋음
  3. Windows: Windows + Shift + S → 포스터 영역 드래그 → 자동 클립보드 저장
  4. Mac: Command + Shift + 4 → 포스터 드래그 → 데스크탑 저장
  5. 또는 브라우저에서 Ctrl + P → "PDF로 저장" 선택
  6. 저장한 이미지를 인스타그램에 업로드!
